# 💬 Real‑Time Chat Application (Frontend)

A modern **Real‑Time Messaging Application** that allows users to chat instantly, send images, and communicate seamlessly. Users can create an account using **Email & Password** or social authentication providers like **Google, GitHub, and Facebook**.

Built with **React + Firebase + Socket.io**, this application delivers fast, secure, and smooth real‑time messaging experience.

# ⚙️ Environment Variables

Create a `.env` file in the root directory:

```
VITE_API_KEY=your_api_key
VITE_AUTH_DOMAIN=your_auth_domain
VITE_PROJECT_ID=your_project_id
VITE_STORAGE_BUCKET=your_storage_bucket
VITE_MESSAGING_SENDER_ID=your_sender_id
VITE_APP_ID=your_app_id
VITE_MEASUREMENT_ID=your_measurement_id

VITE_SOCKET_URL=your_backend_api_url
VITE_BACK_URL=your_backend_api_url/api

cloudinary_name=your_cloudinary_name
cloudinary_key=your_cloudinary_key
cloudinary_secrect=your_cloudinary_secret
```

# 🚀 Getting Started

## 📦 Frontend Setup

### Step 1 — Clone Repository

```
git clone https://github.com/mgrahul63/Real-time-Chat-Frontend
cd Real-time-Chat-frontend
```

---

### Step 2 — Install Dependencies

```
npm install
```

---

### Step 3 — Run Development Server

```
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

---

# ⚙️ Backend Setup

### Step 1 — Clone Repository

```
git clone https://github.com/mgrahul63/Real-time-Chat-Backend
cd Real-time-Chat-Backend
```

---

### Step 2 — Install Dependencies

```
npm install
```

---

### Step 3 — Run Server

```
npm start
```

Backend runs at:

```
http://localhost:4000
```
